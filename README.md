Simple Craps game in Python. Roll two dice and see if you win or lose

## Features
- Rolls two dice and shows the sum
- Follows Craps rules:
  - 7 or 11 on first roll - You win
  - 2, 3, 12 on first roll - You lose
  - Any other sum becomes goal number, roll until goal or 7
- Uses random numbers
